Voice Transcription (Pipeline 1)

Converts audio input to text using Groq's Whisper Large-v3 model.

def transcribe_audio(audio_bytes: bytes, ext: str = "webm") -> str:
    client = Groq(api_key=GROQ_API_KEY)
    response = client.audio.transcriptions.create(
        model="whisper-large-v3",
        file=(f"audio.{ext}", audio_file, f"audio/{ext}"),
    )
    return response.text.strip()  # Clean English text

Logic: Creates a temporary file from audio bytes, sends to Groq API, returns transcribed text. Supports WebM, WAV, MP3, and other formats.

NLP Preprocessing (Pipeline 2)

def full_text_pipeline(raw_text: str) -> dict:
    # Step 1: Language Detection
    detector = _load_lang_detector()  # XLM-RoBERTa
    detection = detector(text[:512])[0]
    lang = detection["label"]  # 'ar', 'fr', 'en', etc.
    
    # Step 2: Translation (if needed)
    if lang != "en" and lang in TRANSLATION_MODELS:
        tokens = tokenizer(text, max_length=512, truncation=True)
        translated = model.generate(**tokens)
        english_text = tokenizer.decode(translated[0])
    
    # Step 3: Stopword Removal & Lemmatization
    cleaned = advanced_preprocess(english_text)
    # Removes: "the", "is", "a" → keeps meaningful words
    # Lemmatization: "visiting" → "visit", "pyramids" → "pyramid"
    
    return {"query_for_rag": cleaned}

Logic: XLM-RoBERTa detects language, Helsinki-NLP translates to English if needed, NLTK lemmatizer reduces words to base form, irregular verb dictionary catches edge cases. Stopword removal eliminates noise.

LLM Generation (Pipeline 5)

Generates expert tour guide answers using Groq's hosted LLMs with retrieved context.

def groq_chat(model: str, messages: list[dict]) -> str:
    client = Groq(api_key=GROQ_API_KEY)
    
    # Messages format: [
    #   {"role": "system", "content": "You are an Egyptian tour guide..."},
    #   {"role": "user", "content": "Query with context + chunks"},
    #   {"role": "assistant", "content": "Previous answer"},
    #   ...
    # ]
    
    response = client.chat.completions.create(
        model="llama-3.3-70b-versatile",  # Or 8B, Mixtral, Gemma
        messages=messages,
        max_tokens=1024,
        temperature=0.7
    )
    return response.choices[0].message.content

Logic: OpenAI-compatible API to Groq endpoint. System prompt sets tour-guide persona. Retrieved context prepended to query. Maintains conversation history for follow-up questions. Model selection: 70B for quality, 8B for speed.

Audio Synthesis & Music Mixing (Pipeline 6)

Converts text to speech and blends with Egyptian ambient music.

def run_tts_pipeline(text: str) -> bytes:
    # Step 1: Clean markdown and send to ElevenLabs
    clean = re.sub(r"[*_`#>~]+", "", text)  # Remove formatting
    
    client = ElevenLabs(api_key=ELEVENLABS_API_KEY)
    gen = client.text_to_speech.convert(
        text=clean,
        voice_id="JBFqnCBsd6RMkjVDRZzb",
        output_format="pcm_24000"  # Raw int16, 24kHz mono
    )
    voice_pcm = np.frombuffer(b"".join(gen), dtype=np.int16)
    
    # Step 2: Generate music bed (or load egyptian_crop.mp3)
    music = _synth_egyptian_music(n_samples, sr=24000)
    # 4 layers: 55Hz drone + Oud partials + Tabla pulses + Shimmer
    
    # Step 3: Mix voice over music
    intro_n = int(3.0 * 24000)  # 3 second intro (music only)
    music[intro_n:intro_n+len(voice)] += voice  # Additive mix
    
    # Step 4: Fade-out outro
    fade = np.linspace(1.0, 0.0, int(3.0 * 24000))  # 3 sec fade
    music[-len(fade):] *= fade
    
    # Step 5: Encode as WAV
    return _pcm_to_wav((music * 32767).astype(np.int16), sr=24000)

Logic: ElevenLabs generates raw PCM at 24kHz (no MP3 decoding overhead). Music synthesized with numpy oscillators (drone, oud, tabla, shimmer) or loaded from file. Voice overlaid additively in middle, 3s intro before voice starts, 3s fade-out at end. All encoded as WAV using stdlib wave module.
